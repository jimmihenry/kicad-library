Hi, these files are ment to enlarge your KiCAD library.

This is the windows path to the KiCAD library "C:\Program Files\KiCad\share\kicad\modules"
	there you will find folders named "*.pretty" in these folders are files withe the
	file extension "*.kicad_mod" these are footprints files.
Example: C:\Program Files\KiCad\share\kicad\modules\Battery.pretty\Battery_CR1225.kicad_mod

	also in the "modules" folder is the "packages3d" folder, there you will find folders
	named "*.3dshapes" in these folders are files withe the file extension "*..step" and
	"*.wrl" these are 3D model files.
Exampel: C:\Program Files\KiCad\share\kicad\modules\packages3d\Battery.3dshapes\Battery_CR1225.step
	 C:\Program Files\KiCad\share\kicad\modules\packages3d\Battery.3dshapes\Battery_CR1225.wrl

For the Heatsink_TO220-3_30x25x25-LM7805 you wont to store the files in respect to the described
path structure.

The Heatsink_TO220-3_30x25x25.3dshapes\Heatsink_TO220-3_30x25x25.step and Heatsink_TO220-3_30x25x25.wrl
use the footprint Heatsink_TO220-3_30x25x25.pretty\Heatsink_TO220-3_30x25x25-assembly.kicad_mod

The Heatsink_TO220-3_30x25x25-LM7805.3dshapes\Heatsink_TO220-3_30x25x25-LM7805.step and
Heatsink_TO220-3_30x25x25-LM7805.wrl both use the footprint 
Heatsink_TO220-3_30x25x25.pretty\Heatsink_TO220-3_30x25x25-assembly.kicad_mod


Example: C:\Users\user\Documents\KiCAD Files\MyKiCADLibrary\Heatsink_TO220-3_30x25x25\Heatsink_TO220-3_30x25x25.3dshapes
	 C:\Users\jimmi\Documents\KiCAD Files\MyKiCADLibrary\Heatsink_TO220-3_30x25x25\Heatsink_TO220-3_30x25x25-LM7805.3dshapes
	 C:\Users\user\Documents\KiCAD Files\MyKiCADLibrary\Heatsink_TO220-3_30x25x25\Heatsink_TO220-3_30x25x25.pretty
